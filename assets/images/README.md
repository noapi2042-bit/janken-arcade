# Character image slots

Optional CPU character images can be dropped here with these names:

- `character_normal.png`
- `character_win.png`
- `character_lose.png`
- `character_draw.png`
- `character_happy.png`
- `character_smug.png`
- `character_worried.png`
- `character_panic.png`
- `character_excited.png`
- `character_shocked.png`

If an image is missing, the game keeps using the built-in fallback character.
